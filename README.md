BMI Calculator
Overview

The BMI Calculator is a simple program that helps users calculate their Body Mass Index (BMI) based on their weight and height. BMI is a measure of body fat based on height and weight and can help assess if a person is underweight, normal weight, overweight, or obese.

This tool is designed to be easy to use, giving instant feedback on your BMI category.

Features

Calculates BMI using weight (kg) and height (m or cm).

Displays BMI value rounded to two decimal places.

Shows BMI category:

Underweight

Normal weight

Overweight

Obese

The user inputs their weight in kilograms.
